1. 모든 소스파일은 UTF-8 인코딩으로 저장

2. import에 (*) 사용하지 않는다

3. 내용이 비어 있거나 single statement라도 중괄호 사용

4. 중괄호 열기 다음에 statement는 다음 줄에 표기

5. 중괄호 닫기는 마지막 statement 이후 다음 줄에 사용

6. 열은 100 자 까지 허용

7. C 스타일의 배열 선언 하지 않는다
   ex) String[] args O, String args[] X

8. switch문에 default를 반드시 사용

9. 들여쓰기는 4개의 공백문자 사용

10. if, while, for or catch 뒤 공백문자 사용 후 괄호

11. 괄호안 , ; 뒤에 공백문자 사용

12. Class 이름 첫글자는 대문자

13. 변수, Method 이름 첫글자 소문자

14. Class, Method, 변수 이름의 구분은 대문자로 한다

15. constant는 대문자로 표기

16. 각 메서드 선언 이후 다음 줄은 공백
